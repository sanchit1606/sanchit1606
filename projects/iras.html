Face Recognition-Based Attendance System

Domain: Computer Vision

Publication: Published in Springer, series: Advances in Information Communication Technology and Computing (AICTC 2024)

Objective

Implemented OpenCV-based facial recognition using Haar cascade and LBPH for classroom attendance automation.

Key Features

  • Real-time face detection using Haar cascade classifier
  • Face recognition using Local Binary Pattern Histogram (LBPH)
  • Automated attendance tracking and logging
  • User-friendly interface for classroom management
  • Database integration for attendance records

Technologies Used

  • Python
  • OpenCV
  • Haar Cascade Classifier
  • LBPH Algorithm
  • SQLite/MySQL for database

Impact

  • Automated attendance tracking reducing manual effort
  • Improved accuracy in attendance records
  • Real-time monitoring capabilities
  • Published research work in international conference
